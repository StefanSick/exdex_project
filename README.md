# exdex_project
## Data Sources

The following economic indicators were sourced from Eurostat:

| Indicator | Description | Dataset |
| :--- | :--- | :--- |
| **Gini Coefficient** | Wealth distribution statistics | [View Data](https://ec.europa.eu/eurostat/databrowser/view/icw_sr_05/default/table?lang=en) |
| **Real GDP** | Real GDP per capita | [View Data](https://ec.europa.eu/eurostat/databrowser/view/sdg_08_10/default/table) |
| **HICP** | Annual data (average index and rate of change) | [View Data](https://ec.europa.eu/eurostat/databrowser/view/prc_hicp_aind/default/table?lang=en) |
| **Poverty Risk** | People at risk of poverty or social exclusion | [View Data](https://ec.europa.eu/eurostat/databrowser/view/ilc_peps01n__custom_19188271/default/table?page=time:2024) |
| **Unemployment Rate** | Unemployment rates by sex, age, and citizenship | [View Data](https://ec.europa.eu/eurostat/databrowser/view/lfsa_urgan/default/table?lang=en&category=labour.employ.lfsa.lfsa_unemp) |
